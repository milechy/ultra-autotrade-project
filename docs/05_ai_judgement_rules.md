# 05_ai_judgement_rules.md  
Ultra AutoTrade – AI判定ルール（具体化版）

---

# 1. BUY 判定条件

```
センチメントスコア > 0.7  
AI信頼度 > 75  
ニュースソースランク A以上  
過去24時間以内にポジティブニュース ≥ 2件  
直近5件の過去ニュースとの整合性が高い  
```

---

# 2. SELL 判定条件

```
センチメントスコア < -0.6  
価格暴落の兆候（ニュース・流動性低下）  
AI信頼度 > 70  
重要ソースからのネガティブ情報  
```

---

# 3. HOLD 判定

```
複数ソースの判断が矛盾  
センチメント -0.3〜0.3 の間  
AI信頼度 < 60  
```

---

# 4. 過剰取引防止

```
1時間以内に同じ判定が3連続 → HOLD
前回取引から10分以内 → HOLD
```

---

# 5. 多数記事の集計ロジック

- スコアを重みづけ合計  
- 信頼度と組み合わせて最終判定  

---

# 6. 判定の最終保証

すべて異常値（信頼度 < 40）の場合 → **HOLD（安全優先）**
